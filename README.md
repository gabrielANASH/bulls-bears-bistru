# Bulls & Bears Bistro - Portfolio Demo

This repo is a personal frontend project I built for my portfolio.
It is inspired by Bulls & Bears Bistro, a real cafe in Coimbatore.

## Important Disclaimer

This is an unofficial concept site made for learning and portfolio showcase.
The actual shop already has its own official website.
This project is not affiliated with or endorsed by Bulls & Bears Bistro.

## Why I Built This

I wanted to practice turning a business-style website into a stronger modern experience.
The focus was on:

- clearer section flow
- better mobile responsiveness
- stronger visual identity
- more engaging interactions
- cleaner component structure

## What Is Inside

- Animated hero section with market-style visuals
- Modular sections for about, menu, specials, reviews, quiz, and location
- Tailwind based design system with custom brand tokens
- Framer Motion animations
- TypeScript + Next.js App Router setup

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
git clone https://github.com/gabrielANASH/bulls-bears-bistru.git
cd bulls-bears-bistru
npm install
npm run dev
```

Open http://localhost:3000

## Production Build

```bash
npm run build
npm start
```
