### Interview Talking Points

**"Tell me about a project you're proud of"**

*"I completely redesigned Bulls & Bears Bistro's website as a portfolio project. 
I identified UX pain points like poor mobile responsiveness and unclear CTAs, 
then built a modern, conversion-focused solution using Next.js, TypeScript, 
and Tailwind CSS. The result achieved 95+ Lighthouse scores and 100% mobile 
responsiveness. What I'm most proud of is the design system I created—it's 
reusable and scalable. This project shows my ability to think about user 
experience, performance optimization, and code quality."*

**"How do you approach performance optimization?"**

*"On this project, I focused on several areas: image optimization using 
Next.js Image component, code splitting with dynamic imports, CSS optimization 
with Tailwind, and animation performance with Framer Motion. I also ran 
bundle analysis to identify code bloat. The result was a 95+ Lighthouse 
score with sub-2-second load times."*

**"How do you ensure code quality?"**

*"I use TypeScript for type safety, organize components in a feature-based 
structure, create reusable hooks, centralize constants and APIs, and maintain 
consistent naming conventions. I also use ESLint and Prettier. On this project, 
every component is self-contained, testable, and well-documented."*

**"Tell me about your design thinking process"**

*"I started by understanding the business goals and user pain points. For 
this restaurant website, I identified key issues: lack of mobile support, 
unclear CTAs, no trust signals. I designed solutions: responsive layout, 
strategic CTA placement, review/rating system. Then I implemented with a 
design system approach—consistent spacing, typography, colors—making future 
maintenance easy."*
